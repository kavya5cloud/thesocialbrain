# TheSocialBrain Landing Page

This is a Next.js app prepared for Vercel deployment.

## Local Run

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Go to [vercel.com/new](https://vercel.com/new).
3. Import your GitHub repository.
4. Framework Preset: `Next.js` (auto-detected).
5. Build Command: `npm run build`
6. Output Directory: leave empty.
7. Install Command: `npm install`
8. Click **Deploy**.

## Notes

- The site copy uses early-stage, realistic traction numbers suitable for a startup launch.
- TypeScript build errors are no longer ignored in production config.
